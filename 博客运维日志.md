## 一、项目概览

纯静态个人博客，部署于 GitHub Pages，后端数据存储接入 Supabase 云数据库，支持多设备数据互通。

### 技术栈

| 层 | 技术 |
|---|---|
| 前端 | HTML + CSS + JavaScript（原生，无框架） |
| 部署 | GitHub Pages（免费 HTTPS + 全球 CDN） |
| 云数据库 | Supabase PostgreSQL（免费 500MB） |
| JS 库 | supabase-js v2（CDN 引入） |
| 图标 | Font Awesome 6.4.0（CDN 引入） |

### 文件结构

```
blog/
├── index.html       # 页面结构
├── style.css        # 完整样式（双主题 + 响应式）
└── script.js        # 交互逻辑 + Supabase 对接
```

## 五、🚨 关键 Bug 修复记录

### Bug 1：登录按钮失效

`let supabase` 与 CDN 全局变量冲突 → 改名为 `dbClient`

### Bug 2：云端写入失败

JS 字段 `name`/`userId` 与 SQL 列 `username`/`user_id` 不匹配 → 全部对齐

### Bug 3：ID 溢出

`Date.now()` (13 位) 超 INTEGER 范围 → `genSafeId()` 控制到 2,000,000,000 以内
